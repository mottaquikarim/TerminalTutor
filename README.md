# TerminalTutor

terminal documentation accessible via commandline.

### move file, rename file

* rename or move file1 to file2
* if file2 is an existing directory, moves file1 into directory file2

```bash
$ mv file1 file2
```

### copy file

* copy file1 to file2

```bash
$ cp file1 file2
```
### copy folder

* copy dir1 to dir2
* create dir2 if it doesn't exist
```bash
$ cp -r dir1 dir2
```
